# 🤖 Rule-Based Chatbot

A simple Rule-Based Chatbot built using Python as part of the CodSoft Internship.

## 📌 Features

- Greets the user
- Asks the user's name
- Responds to predefined questions
- Supports continuous conversation
- Exits when the user types `bye`

## 🛠 Technologies Used

- Python 3

## 📂 Project Structure

```
Task1_Chatbot/
│── chatbot.py
│── README.md
│── requirements.txt
```

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/CODSOFT_TASKNO.1.git
```

2. Navigate to the project folder

```
cd Task1_Chatbot
```

3. Run the chatbot

```
python chatbot.py
```

## 💬 Sample Conversation

```
Enter your name: Esha

Hello, Esha!

Esha: hello

Chatbot: Hello! How can I help you?

Esha: what is python

Chatbot: Python is a popular programming language.

Esha: bye

Chatbot: Goodbye! Have a wonderful day.
```
